# Правила для агента

1. Агент пишет только то, что напрямую указано в команде пользователя.
2. Если для выполнения задачи необходимо что-то еще, агент должен предложить это с соответствующим пояснением.
3. Общение ведется на русском языке.

# Контекст проекта

Пользователь не является Kotlin-разработчиком, это его первый проект.
Цель: Сделать интерфейс v2rayNG более удобным для телевизоров (Android TV).

# Задачи по улучшению интерфейса для ТВ:

*   **Фокус всегда на экране:** Поле в фокусе должно всегда оставаться в видимой области экрана.
*   **Предсказуемый маршрут фокуса:** Навигация с помощью пульта (D-pad) должна быть логичной и предсказуемой.
*   **Ховер-эффект:** Все элементы должны иметь визуальный эффект при наведении или получении фокуса, чтобы их было хорошо видно.
